# Imágenes de Productos

## Ubicación
Esta carpeta contiene todas las imágenes de los productos de MediaDaw.

## Nomenclatura
Las imágenes deben seguir el formato: `nombre-producto.jpg` o `nombre-producto.png`

Ejemplos:
- `iphone-15-pro.jpg`
- `macbook-air-m2.jpg`
- `gopro-12.jpg`
- `nintendo-switch.jpg`

## Imagen por Defecto
Si una imagen de producto no se encuentra, el sistema automáticamente mostrará el logo de MediaDaw como imagen por defecto.

## Formato Recomendado
- **Formato**: JPG o PNG
- **Tamaño**: 800x800 px (1:1)
- **Peso**: Máximo 500 KB
- **Fondo**: Blanco o transparente

## Cómo Añadir Imágenes
1. Coloca la imagen en esta carpeta
2. Nombra el archivo según el producto
3. Actualiza el campo `imagen` del producto en la base de datos con el nombre del archivo
4. La ruta completa será: `/images/productos/nombre-archivo.jpg`
